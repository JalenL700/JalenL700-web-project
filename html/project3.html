Welcome to the game

This webpage is about my favorite horror game

Welcome to the game is a psychological horror series developted by reflect studios, the game involves exploring the dark web to find keys to a red room: All while dodging hacks and killers trying to get to you.

This series has multiple games, which are listed below.

  1. Welcome to the game
  2. Welcome to the game 2
  3. Scrutinized
  4. Dead signal
  5. Welcome to the game 3(unreleased)

This game is one of my favorites, I mostly enjoy it because you are on a computer and it involves hacking and the dark web.
